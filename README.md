Projet : Gestion de bibliothèque

Technologies utilisées :
- Laravel
- PHP
- MySQL
- HTML / CSS

Fonctionnalités :
- Authentification
- Gestion des livres
- Gestion des utilisateurs
- Réservations

Installation :
1. composer install
2. Copier .env.example en .env
3. php artisan key:generate
4. Configurer la base de données
5. php artisan migrate
6. php artisan serve
